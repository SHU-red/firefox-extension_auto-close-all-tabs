# Auto Close All Tabs

A simple Firefox extension that closes all tabs after a specified period of inactivity.

## Installation

1.  Clone this repository or download the files.
2.  Open Firefox and navigate to `about:debugging`.
3.  Click on "This Firefox" in the sidebar.
4.  Click on "Load Temporary Add-on...".
5.  Select the `manifest.json` file from the cloned/downloaded directory.

## Usage

1.  Right-click the extension icon in the toolbar and select "Options".
2.  Enter the desired inactivity timeout in seconds.
3.  Click "Save".

The extension will now close all tabs after the specified period of inactivity. A new tab will be created to keep the browser window open.
